### Overview
    A simple weather application that allows users to search for a city
    and view current weather information with a short forecast.
### Features
    - Search weather by city name
    - Display temperature and weather condition
    - Show humidity and wind speed
    - Error handling for invalid cities
    - 3-day weather forecast
### Technologies
    - HTML5
    - CSS3
    - JavaScript (Vanilla JS)
    - Weather API
### What I Learned
    - Working with a public weather API
    - Using `fetch` and `async/await`
    - Handling errors in JavaScript
    - Updating the UI with dynamic data
